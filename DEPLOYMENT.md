# 🚀 Развёртывание менеджера задач

## 📋 Системные требования

### Обязательно установите:
- **Python 3.11+** (https://www.python.org/downloads/)
- **Node.js** (https://nodejs.org/)

### Проверка установки:
```cmd
python --version
node --version
```

---

## ⚡ Первый запуск (один раз)

1. **Распакуйте** папку проекта в удобное место
2. **Откройте Command Prompt** в папке проекта
3. **Запустите:**
   ```cmd
   setup_project.bat
   ```
   Это автоматически установит все зависимости и подготовит базу данных.

---

## 🚀 Обычный запуск (каждый день)

```cmd
start_all.bat
```

Откроются 2 окна и браузер автоматически откроется на http://localhost:5173

## 📚 Доступ к приложению

- **Основной интерфейс:** http://localhost:5173
- **API документация:** http://localhost:8000/docs
- **Backend API:** http://localhost:8000

---

## ⚙️ Автозапуск при включении ПК

1. Откройте PowerShell **как администратор**
2. Выполните:
   ```powershell
   & "путь\к\проекту\setup_autostart.ps1"
   ```
3. После перезагрузки приложение будет запускаться автоматически

---

## 🛠️ Возможные проблемы

### "Python не найден"
Установите Python и отметьте "Add Python to PATH" при установке.

### "Node.js не найден"
Установите Node.js с https://nodejs.org/

### "Port 5173 already in use"
Закройте предыдущее окно frontend или перезагрузите ПК.

### Backend не запускается
Дождитесь 5 секунд после запуска `start_all.bat`.
