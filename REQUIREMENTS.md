# 📋 Системные требования

## Минимум для работы

### Для любого ПК
- **Windows 10/11** (64-bit)
- **4 ГБ оперативной памяти** (для комфортной работы)
- **500 МБ свободного места** на диске

### Что нужно установить

#### 1. Python 3.11+
**Скачать:** https://www.python.org/downloads/
- Выберите последнюю версию Python 3.11 или 3.12
- **ВАЖНО!** При установке отметьте `Add Python to PATH`
- Нажмите "Install Now"

**Проверка:**
```cmd
python --version
```

#### 2. Node.js + npm
**Скачать:** https://nodejs.org/
- Выберите LTS версию (рекомендуется)
- Нажмите "Далее" и "Установить"

**Проверка:**
```cmd
node --version
npm --version
```

#### 3. Git (опционально, но рекомендуется)
**Скачать:** https://git-scm.com/download/win
- Используйте стандартные параметры при установке

---

## Проверка установки

Откройте **Command Prompt** (или PowerShell) и выполните:

```cmd
python --version
node --version
npm --version
```

Все три команды должны показать версии без ошибок.

---

## Установка проекта

1. **Распакуйте** папку проекта в удобное место
2. **Откройте Command Prompt** в папке проекта
3. **Запустите:**
   ```cmd
   setup_project.bat
   ```
   
   Это автоматически установит все зависимости!

4. **После завершения запустите:**
   ```cmd
   start_all.bat
   ```

---

## Что делают скрипты?

| Скрипт | Назначение |
|--------|-----------|
| `setup_project.bat` | Первая установка всех зависимостей (только один раз) |
| `start_all.bat` | Запуск приложения (используется каждый день) |
| `reset_project.bat` | Удаление БД и пересоздание (если что-то сломалось) |
| `create_desktop_shortcut.bat` | Создание ярлыка на рабочем столе для быстрого запуска |
| `setup_autostart.ps1` | Автозапуск приложения при включении ПК |

## Возможные проблемы

### "Python не найден"
**Решение:** Установите Python и при установке отметьте "Add Python to PATH"

### "Node.js не найден"
**Решение:** Установите Node.js с https://nodejs.org/

### "Port 5173 is already in use"
**Решение:** Закройте предыдущее окно frontend или перезагрузитесь

### "Connection refused на http://localhost:8000"
**Решение:** Подождите 5 секунд, пока backend стартует полностью
